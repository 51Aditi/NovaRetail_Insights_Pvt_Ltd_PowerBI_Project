# NovaRetail Insights - Power BI Project

## 📌 Project Overview
NovaRetail Insights is a Power BI data analytics project focused on analyzing retail sales performance, customer behavior, profitability, and business trends using interactive dashboards.

The project transforms raw retail data into meaningful insights to support data-driven decision-making.

## 🎯 Objectives
- Analyze overall sales and profit performance
- Identify top-performing products and categories
- Understand customer segments and purchasing patterns
- Track delivery and order performance
- Find areas for improving business profitability

## 🛠️ Tools & Technologies
- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Data Visualization
- Data Cleaning & Transformation

## 📊 Dashboard Features

### Sales Analysis
- Total Sales
- Total Profit
- Total Orders
- Sales trends over time

### Customer Analysis
- Customer segment performance
- Customer-wise sales contribution
- Buying patterns

### Product Analysis
- Category-wise sales
- Profit by product category
- Best-performing products

### Delivery Analysis
- Shipping mode analysis
- Delivery status tracking
- Late delivery insights

## 📈 Key Insights
- Identified high-performing markets and categories
- Analyzed profit contribution across segments
- Studied impact of discounts on profitability
- Evaluated delivery performance
